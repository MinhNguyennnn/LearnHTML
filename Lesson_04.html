Chapter Summary

  • All HTML elements can have attributes
  • The title attribute provides additional "tool-tip" information
  • the href attribute provides address infomation for links
  • the Width and hight attributes provide size information for images
  • The alt attribute provides text for screen readers
  • At W3Schools we alway use lowercase attribute names
  • At W3Schools we always quote attribute values
